# Project Scout Scripts

This directory contains Python scripts for real data collection and analysis.

## Scripts

### 1. test_config.py
**Purpose:** Verify your API credentials are configured correctly

**Usage:**
```bash
python scripts/test_config.py
```

**What it tests:**
- config.json exists and is valid JSON
- Reddit API credentials work
- GitHub API token is valid
- Hacker News API is accessible
- AI API (Anthropic/OpenAI) is configured

---

### 2. scrape_data.py
**Purpose:** Scrape real discussions from Reddit, HN, and GitHub

**Usage:**
```bash
python scripts/scrape_data.py
```

**What it does:**
1. Connects to Reddit, GitHub, HN APIs
2. Searches for discussions about pain points and needs
3. Filters by date (last 3 months by default)
4. Saves raw data to `scraped_data.json`

**Output:**
- `scraped_data.json` - Raw discussions with metadata

**Time:** 2-5 minutes depending on API rate limits

---

### 3. analyze_data.py
**Purpose:** Extract pain points from scraped data

**Usage:**
```bash
python scripts/analyze_data.py
```

**What it does:**
1. Reads `scraped_data.json`
2. Extracts keywords and themes
3. Clusters discussions by topic
4. Ranks pain points by potential
5. Saves structured pain points to `pain_points.json`

**Output:**
- `pain_points.json` - Structured pain points with scores

---

## Workflow

**Complete workflow to generate a real Project Scout report:**

```bash
# Step 1: Set up configuration
cp config.example.json config.json
nano config.json  # Add your API keys

# Step 2: Test configuration
python scripts/test_config.py

# Step 3: Scrape real data
python scripts/scrape_data.py

# Step 4: Analyze data
python scripts/analyze_data.py

# Step 5: Generate report (in Claude Code)
/scout
```

## Troubleshooting

### "Module not found" errors
```bash
pip install [missing_module]
```

### "Config not found" error
```bash
cp config.example.json config.json
# Then edit config.json with your API keys
```

### Reddit API authentication failed
- Check client_id and client_secret in config.json
- Verify you created a "script" type app, not "web app"
- Make sure user_agent is unique

### GitHub API rate limit
- You get 5,000 requests/hour with a token
- Enable caching in config.json
- Wait for rate limit reset (check with test_config.py)

### No data scraped
- Check your search keywords in config.json
- Extend time_range_months (try 6 instead of 3)
- Some subreddits may have low activity

## Advanced Usage

### Custom time range
Edit `config.json`:
```json
{
  "settings": {
    "time_range_months": 6  // Changed from 3 to 6
  }
}
```

### More discussions per source
```json
{
  "settings": {
    "max_discussions_per_source": 100  // Changed from 50
  }
}
```

### Additional subreddits
```json
{
  "settings": {
    "subreddits": [
      "SomebodyMakeThis",
      "AppIdeas",
      "Entrepreneur",
      "SideProject",
      "startups",      // Add more
      "indiehackers"
    ]
  }
}
```
